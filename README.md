# 🏦 Banking Management System in C

## 📌 Project Overview

The **Banking Management System** is a console-based application developed in **C Programming Language** as part of the **CodeAlpha Virtual Internship Program**.

This project simulates basic banking operations such as **Account Creation, Deposit, Withdrawal, and Balance Enquiry**. It uses **Structures and Functions** to manage account information efficiently and provides a simple menu-driven interface for user interaction.

The project helps in understanding fundamental programming concepts including structures, functions, data handling, and modular programming in C.

## 🚀 How to Run

### Step 1: Clone Repository

```bash
git clone https://github.com/your-username/banking-management-system.git
```

### Step 2: Navigate to Project Folder

```bash
cd banking-management-system
```

### Step 3: Compile

```bash
gcc banking_system.c -o banking
```

### Step 4: Run

#### Windows

```bash
banking.exe
```

#### Linux / Mac

```bash
./banking
```
